# hubot-groot

One groot at a time.GIFs.

See [`src/groot.coffee`](src/groot.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-groot --save`

Then add **hubot-groot** to your `external-scripts.json`:

```json
["hubot-groot"]
```

## Sample Interaction

```
user1>> hubot groot
hubot>> [gif-caption]
hubot>> [gif-url]
```
